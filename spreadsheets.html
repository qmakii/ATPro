Электронные таблицы (Excel / Google Sheets)

📈 Основы работы

Электронная таблица состоит из ячеек, расположенных на пересечении строк и столбцов. Адрес ячейки: буква столбца + номер строки (например, B4).

A B C
1 Товар Цена Количество
2 Ноутбук 50000 5
3 Мышь 1500 20

🧮 Формулы и функции

Формула всегда начинается со знака =. Основные функции:

  • =СУММ(A1:A10) — сумма чисел в диапазоне
  • =СРЗНАЧ(B1:B5) — среднее арифметическое
  • =ЕСЛИ(A1>10; "Да"; "Нет") — условный оператор
  • =ВПР(E1; A1:B10; 2; ЛОЖЬ) — вертикальный просмотр
Пример: =B2*C2 → вычисляет стоимость (цена × количество)

📊 Абсолютные и относительные ссылки

Относительная (A1): изменяется при копировании формулы.

Абсолютная ($A$1): фиксирует столбец и строку (не меняется).

Смешанная ($A1 или A$1): фиксирует только столбец или только строку.

// Пример использования абсолютной ссылки для налоговой ставки
= B2 * $E$1  // где в E1 хранится ставка НДС (например, 0.20)

📉 Диаграммы и сводные таблицы

Сводная таблица — мощный инструмент для агрегации и анализа больших объёмов данных (группировка, сумма, среднее).

Диаграммы: гистограммы, круговые, линейчатые — для визуализации трендов и сравнений.

Совет: Используйте Условное форматирование для автоматической подсветки ячеек по правилам (например, все продажи >1000 зелёным цветом).
