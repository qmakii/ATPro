Кодирование информации

📝 Текстовое кодирование

ASCII — 7-битная кодировка (0-127), содержит латиницу, цифры, знаки препинания.

Unicode (UTF-8, UTF-16) — поддерживает символы всех языков мира, эмодзи.

Символ ASCII (дес.) ASCII (двоич.) UTF-8 (hex)
A 65 01000001 41
Я - - D0 AF
$ 36 00100100 24

🖼️ Кодирование изображений

Растровая графика: изображение состоит из пикселей. Каждый пиксель кодируется цветом (RGB, CMYK).

Формула объёма: V = ширина × высота × глубина цвета (бит на пиксель)

// Пример расчёта размера изображения
ширина = 1920
высота = 1080
глубина_цвета = 24  # бит (True Color)
объём_бит = 1920 * 1080 * 24
объём_Мбайт = объём_бит / 8 / 1024 / 1024
print(f"Размер: {объём_Мбайт:.2f} МБ")  # ~5.93 МБ

🎵 Кодирование звука

АЦП (аналогово-цифровой преобразователь) — частота дискретизации (Гц) и разрядность квантования.

Формула объёма звукового файла: V = частота × разрядность × время × количество каналов

# Пример для стерео (2 канала), 44.1 кГц, 16 бит, 60 секунд
частота = 44100
разрядность = 16  # бит
время_сек = 60
каналы = 2
объём_бит = частота * разрядность * время_сек * каналы
объём_Мбайт = объём_бит / 8 / 1024 / 1024
print(f"Размер: {объём_Мбайт:.1f} МБ")  # ~10.1 МБ

🔐 Кодирование для передачи данных

Base64 — преобразование двоичных данных в текст (ASCII). Используется в email, URL, API.

# Python: кодирование в Base64
import base64
text = "Hello, мир!"
encoded = base64.b64encode(text.encode('utf-8'))
print(encoded)  # b'SGVsbG8sINC80LjRgCE='

decoded = base64.b64decode(encoded).decode('utf-8')
print(decoded)  # Hello, мир!
