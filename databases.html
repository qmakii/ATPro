Базы данных (SQL и NoSQL)

📊 Что такое база данных?

База данных (БД) — организованная структура для хранения, управления и извлечения данных. СУБД — система управления базами данных (MySQL, PostgreSQL, SQLite, MongoDB).

🗄️ Реляционные БД (SQL)

Данные хранятся в таблицах, связанных ключами. Первичный ключ — уникальный идентификатор записи. Внешний ключ — ссылается на первичный ключ другой таблицы.

Таблица "Студенты"
id (PK) имя курс
1 Анна 3
2 Борис 2 
-- Создание таблицы
CREATE TABLE students (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    course INTEGER
);

-- Вставка данных
INSERT INTO students VALUES (1, 'Анна', 3);

-- Запрос с JOIN
SELECT s.name, g.grade
FROM students s
JOIN grades g ON s.id = g.student_id
WHERE g.subject = 'Математика';

📁 Нереляционные БД (NoSQL)

Для больших объёмов неструктурированных данных: MongoDB (документы), Redis (ключ-значение), Neo4j (графы).

// Документ MongoDB
{
  "_id": ObjectId("507f1f77bcf86cd799439011"),
  "username": "ivan_petrov",
  "email": "ivan@example.com",
  "courses": ["Алгоритмы", "Базы данных"],
  "profile": {
    "age": 20,
    "city": "Москва"
  }
}

🔗 Нормализация

Устранение избыточности и аномалий:

  • 1НФ: атомарные значения, нет повторяющихся групп
  • 2НФ: полная зависимость от первичного ключа
  • 3НФ: отсутствие транзитивных зависимостей
